# Personal Use API Hub

A full-stack SaaS developed web application that allows users to connect external API's, visualize data, and allows them to automate workflows based on API responses.

-------------------------------------------------------------

## Overview

Usage:

- Connect any REST API
- See and interact with API responses
- Query / sort by specific data fields
- Visualize API responses via graphs / dashboards
- Automate emails/texts/ other flags based on data recieved

Example Use Cases:

- Monitoring weather
- Tracking stock prices
- Recording social media activity (likes/reposts/etc..)

-------------------------------------------------------------
